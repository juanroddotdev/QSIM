# QSIM Documentation

This folder contains documentation diagrams for the QSIM Shopify Private App.

## Files

- **ARCHITECTURE.md** - System Architecture diagram
- **DATA_FLOW.md** - Data Flow diagram  
- **USER_WORKFLOW.md** - User Workflow diagram
- **index.html** - Local viewer for all diagrams (open in browser)

## Viewing Diagrams

### On GitHub/GitLab
The markdown files (`.md`) will automatically render Mermaid diagrams when viewed on GitHub or GitLab.

### Locally
1. Open `index.html` in your web browser
2. The diagrams will render using Mermaid.js from CDN

## Adding Your Diagrams

Replace the placeholder Mermaid code blocks in each `.md` file with your actual diagram code.
